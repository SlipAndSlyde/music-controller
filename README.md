# READ ME

## Introduction
Disclaimer: I am a self-taught programming hobbyist.

The objective of this program is to have a simple, practical and flexible music controller class that can be implemented in other projects.

## Features
Current features:
- play music
- pause music
- continue music
- stop music

Upcoming features:
- revamp design
- volume control
- playlists
- autoplay
- shuffle playlist
